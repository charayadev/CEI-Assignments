# CEI Pandas Data Cleaning Project

## Overview

This project was completed as part of the **Celebal Excellence Internship – Week 1 Project**.  
The main objective of this project was to learn Python basics and perform data exploration and cleaning using the Pandas library.

The project demonstrates the complete workflow of:
- Loading a dataset
- Exploring and understanding the data
- Handling missing values
- Performing basic data operations
- Checking duplicate records
- Creating a cleaned dataset
- Exporting the final processed data

# Project Workflow

## 1. Data Loading
- Loaded the shopping dataset into a Pandas DataFrame.

## 2. Data Exploration
Performed:
- `head()`
- `tail()`
- dataset shape analysis
- column inspection
- datatype checking

## 3. Missing Value Handling
- Identified missing values in the dataset
- Filled numerical and categorical missing values using appropriate techniques
- Removed unnecessary high-missing-value columns where required

## 4. Duplicate Analysis
- Checked duplicate records in the dataset
- Verified dataset consistency

## 5. Basic Data Operations
Performed:
- row filtering
- column selection
- dataset inspection

## 6. Data Cleaning
- Cleaned and processed the dataset for better usability
- Improved dataset consistency and readability

## 7. Final Dataset Export
- Exported the cleaned dataset as a new CSV file for future use and analysis

# Brief Summary

In this project, data exploration and cleaning were performed using Python Pandas on a shopping dataset. Missing values were handled appropriately, duplicate records were checked, and basic preprocessing operations were performed. The cleaned dataset was successfully exported as a new CSV file.
